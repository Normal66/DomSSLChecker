## Русский

### 1.01

**Что нового**
- корректный ввод кириллических доменов (IDN) и URL вида `http://xn--.../`
- в UI показывается домен в кириллице и punycode (для проверки/копирования)
- определение “хостера” улучшено: по A/ASN (и часто CDN) после SSL‑проверки
- в деталях домена отображаются **регистратор** и **хостер/NS**

**Файлы**
- APK: `DomSSLChecker-1.01.apk`

## English

### 1.01

**What’s new**
- proper Cyrillic IDN input + support for URL input like `http://xn--.../`
- UI shows Unicode domain + punycode (for verification/copy)
- improved “hoster” detection: based on A/ASN (often CDN) after SSL check
- domain details show **registrar** and **hoster/NS**
